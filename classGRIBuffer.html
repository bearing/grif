Detailed Description

A class used as a data structure for storing member.

The buffer class is an abstraction of the buffer abstract data type. buffer objects will be and should ONLY be stored by the memoryManager. It is not a versatile abstract data type. The buffer class comes implemented with a number of functions that allow read/write access. It also keeps internal state for keeping track of file access and growth.
