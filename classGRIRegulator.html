bool GRIRegulator::SetPacketPosition ( QString  bufferName,
int  packetNumber 
)

SetPacketPosition() sets the packet marker for the specified buffer to the packetNumber position. This is useful for use with the overloaded ReadMemory() function which allows users to read the packet that has been indexed by the packet marker. This is in most cases the last packet to be read from unless SetPacketPosition() has been called. If the operation is successful, it returns true, otherwise false.
