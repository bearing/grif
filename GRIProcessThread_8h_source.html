00053 // GRIProcessThread class is a common interface for DAQ thread and analysis
00054 // thread.
00055 //
00056 // Load balancing is being partially done in here (ie: the actual changing of
00057 // thread's priority). It is done by having two sets of variables. One is the
00058 // number of packets before we are allowed to adjust the priority of this 
00059 // process away/to saturation. Saturation is defined as the time-critical 
00060 // priority or idle priority. The other one is when is the last time this 
00061 // process adjusts its priority away/to saturation. If the latter is greater 
00062 // than the former, priority adjustment will be done.

00269   GRIProcessThread::ReadMemory(QString blockName ,QString bufferName) {
