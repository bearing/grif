void GRIDataBlock::set_is_enabled ( bool  is_enabled  )  [inline]

If the data block is active, then data will pass through this block in the system. If the data block is not active, then data will not pass through this block, and a warning message will be posted. The block's active status can be changed as often as the user wants during run time via GRIProcessThread::EnableDataBlock() and GRIProcessThread::DisableDataBlock(). By default, the data block is enabled for use.
